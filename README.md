# Tripleten web_project_around

Projeto desenvolvido durante a sprint 7

# Descrição do Projeto

Around The U.S. é uma aplicação web interativa que permite aos usuários compartilhar e explorar fotos de lugares incríveis dos Estados Unidos. O projeto apresenta um design moderno e responsivo, com funcionalidades de perfil de usuário e galeria de imagens.

# Tecnologias Utilizadas

HTML
CSS
JavaScript
Metodologia BEM
